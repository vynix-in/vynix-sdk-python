# Contributing to Vynix Python SDK

Thanks for your interest in improving Vynix Python SDK, part of the [Vynix](https://vynix.in) toolkit.

## How to help

- Open an issue for bugs or ideas.
- Send a pull request for fixes and small improvements.
- Improve the documentation.

## Ground rules

- Keep changes focused and well described.
- Be kind in reviews and discussions.

For product feedback about Vynix itself, the fastest path is to use Vynix on your own site at [https://vynix.in](https://vynix.in).
